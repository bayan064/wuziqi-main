# 五子棋技能系统设计文档

本文档基于UI设计图，汇总了游戏中需要实现的所有特殊技能。每个技能包含名称、冷却时间（CD/回合数）以及具体的效果描述。在开发 `utils/skill.js` 时，请参考以下设定进行逻辑实现。

## 技能列表

### 1. 飞沙走石
* **冷却时间**：3回合
* **技能效果**：选择一个对手的棋子扔出棋盘。
* *开发建议*：需要配合交互，玩家点击技能后，处于“选子模式”，点击对方落子点即可将其从 `board` 数组中移除（置为0），并记录被扔走的棋子信息供“拾金不昧”使用。

### 2. 拾金不昧
* **冷却时间**：5回合
* **技能效果**：将被扔走的棋子随机恢复到棋盘。
* *开发建议*：检查是否有被“飞沙走石”移除的本方棋子记录，如果有，在棋盘的空位（值为0的位置）中随机选取一个位置将其恢复。

### 3. 保洁上门
* **冷却时间**：7回合
* **技能效果**：随机清理对手1-3个棋子。
* *开发建议*：获取棋盘上所有对手棋子的坐标，随机打乱后截取1至3个（具体数量可以使用 `Math.random()` 决定），将这些坐标对应的值置为0。

### 4. 擒拿擒拿
* **冷却时间**：8回合
* **技能效果**：下回合克制对手的飞沙走石和保洁上门技能。
* *开发建议*：设置一个状态标志位（如 `isProtected = true`），在对手回合如果对手使用特定技能时，检测本状态并使其技能无效化。

### 5. 静如止水
* **冷却时间**：10回合
* **技能效果**：禁用对手下回合的技能使用。
* *开发建议*：设置一个全局或对手的状态标志位（如 `isSilence = true`），并在UI和逻辑层限制对手下回合点击任何技能按钮。

### 6. 时光倒流
* **冷却时间**：12回合
* **技能效果**：让时间回到上一回合，且不恢复技能CD。
* *开发建议*：需要引入简单的“历史记录”栈功能，每次落子或结算后保存当前的棋盘快照，使用技能时将 `board` 恢复到上一个快照，但保留当前的技能CD状态。

### 7. 两级反转
* **冷却时间**：999回合（视为单局限用极少次/1次）
* **技能效果**：交换棋盘中双方的棋子。
* *开发建议*：遍历整个 `board`，将所有的 1（黑）变成 2（白），将所有的 2（白）变成 1（黑）。

### 8. 力拔山兮
* **冷却时间**：999回合（一局只能使用一次）
* **技能效果**：清空棋盘中所有棋子。
* *开发建议*：遍历整个 `board` 数组，将所有元素重置为0。
